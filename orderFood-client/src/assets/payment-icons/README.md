# 付款方式图标目录

请将对应的品牌logo SVG文件放置在此目录下，文件命名规则：

- `wechat.svg` - 微信支付
- `alipay.svg` - 支付宝
- `visa.svg` - Visa
- `mastercard.svg` - Mastercard
- `octopus.svg` - 八达通
- `cash.svg` - 现金

## 注意事项

1. SVG文件应该使用合适的viewBox，建议使用 `viewBox="0 0 24 24"` 或类似的尺寸
2. 确保SVG文件可以直接在浏览器中正常显示
3. 如果SVG文件有固定颜色，建议保留；如果需要适配主题，可以使用 `currentColor`
