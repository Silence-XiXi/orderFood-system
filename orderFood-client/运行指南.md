# 自助点单系统前端运行指南

## 快速开始

### 1. 安装依赖

```powershell
cd orderFood-client
npm install
```

### 2. 启动开发服务器

```powershell
npm run dev
```

### 3. 访问应用

浏览器会自动打开，或手动访问：
- **本地访问**：`http://localhost:5173`
- **网络访问**：`http://[你的IP]:5173`

## 配置后端地址

### 默认配置

默认后端地址为 `http://127.0.0.1:3002`

### 修改后端地址

**方式一：环境变量（推荐）**

```powershell
$env:VITE_API_TARGET="http://192.168.1.100:3002"
npm run dev
```

**方式二：修改配置文件**

编辑 `vite.config.js`，修改 `API_TARGET` 变量：

```javascript
const API_TARGET = process.env.VITE_API_TARGET || 'http://192.168.1.100:3002';
```

## 构建生产版本

### 构建

```powershell
npm run build
```

构建后的文件在 `dist` 目录中。

### 预览构建结果

```powershell
npm run preview
```

## 使用说明

1. **选择套餐**：点击"一菜套餐"或"两菜套餐"卡片
2. **确认付款**：点击"确认付款"按钮
3. **等待处理**：系统会创建订单并打印小票
4. **完成**：看到成功提示后，可以继续下一个订单

## 注意事项

- 确保后端服务器已启动（`orderFood-server`）
- 如果后端在不同机器上，需要配置正确的后端地址
- 付款功能暂未实现，目前只是模拟流程
